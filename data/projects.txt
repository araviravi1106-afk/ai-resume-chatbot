PROJECT 1: AI Resume Assistant Chatbot (Final Year Project)
------------------------------------------------------------

Description:
Aravindh's final year project is an AI Resume Assistant Chatbot. The chatbot is designed to answer interview questions about him automatically based on his resume and project details. It uses Retrieval Augmented Generation to retrieve relevant information from his resume dataset and generate accurate, professional responses.

Technologies Used:
Python, LangChain, ChromaDB (vector database), Sentence Transformer embeddings, Ollama (local LLM), Streamlit (web interface)

Role: Solo developer — designed the RAG architecture, built the ingestion pipeline, prompt engineering, and deployed the full Streamlit interface.

Key Features:
- Resume data ingestion and chunking
- Embedding generation using Sentence Transformers
- Vector storage and semantic search using ChromaDB
- Prompt engineering to enforce polite interview-style answers
- Local LLM inference using Ollama (no internet required)
- Interactive chat interface using Streamlit
